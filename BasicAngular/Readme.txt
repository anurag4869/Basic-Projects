This folder only contains "src" folder other prerequistes are installed as soon as angular is installed.

In "src" go inside "app" folder.

To create different (such as home page component)
components use: ng g component home
or use: npm run ng g component home

Add neccessary html code to home.component.html

The files to which changes are made are as follows:
	* app.component.html
	* app.module.ts
	* app-routing.module.ts
